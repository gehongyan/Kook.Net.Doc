区域文本 paragraph

属性 类型 默认值 必需 说明
cols integer null ✅ 区域文本的列数，范围为 1-3
元素 数量 说明
plain 与 kmarkdown 合计至多 50，至少 1 区域文本的内容
kmarkdown 与 plain 合计至多 50，至少 1 区域文本的内容 
<paragraph cols="3">
  <plain>Header 1</plain>
  <plain>Header 2</plain>
  <plain>Header 3</plain>
  <kmarkdown>Value 1</kmarkdown>
  <kmarkdown>Value 2</kmarkdown>
  <kmarkdown>Value 3</kmarkdown>
</paragraph>
