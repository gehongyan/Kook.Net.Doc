Properties

SlowModeInterval

获取此频道当前设置的慢速模式延迟。

int SlowModeInterval { get; }

Property Value

int

一个 int，表示用户在可以发送另一条消息之前需要等待的时间（以秒为单位）；如果未启用，则为 0。

Remarks

拥有 ManageMessages 或 ManageChannels 权限的用户不受慢速模式延迟的限制。
