Properties

Creator

获取创建此频道的用户。

public SocketGuildUser? Creator { get; }

Property Value

SocketGuildUser

Remarks

此属性会尝试从缓存的用户列表中获取此频道的创建者。如果用户不是此服务器的成员，或该用户不在本地缓存的用户列表中，则此属性将返回 null。在这种情况下，可以尝试通过 GetUserAsync(ulong, CacheMode, RequestOptions?) 获取指定的用户信息。

Remarks
