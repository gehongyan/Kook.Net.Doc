Enum TagHandling

Namespace
Kook
Assembly
Kook.Net.Core.dll

表示在转换标签时应进行的操作类型。

public enum TagHandling

Fields

FullName = 4

转换为名称，包含提及前缀。如果提及标签为用户提及，则还会包含用户的识别号。

例如：

原始值 转换结果
(met)2810246202(met) @用户名#0001
(chn)5017360261312802(chn) #频道名称
(rol)6790760(rol) @角色名称
(met)all(met) @全体成员
(met)here(met) @在线成员
(emj)kooknet-logo(emj)[1591057729615250/E8fy7hpRFU1jk1jk] :kooknet-logo:
FullNameNoPrefix = 5

转换为名称，但不包含提及前缀。如果提及标签为用户提及，则还会包含用户的识别号。

例如：

原始值 转换结果
(met)2810246202(met) 用户名#0001
(chn)5017360261312802(chn) 频道名称
(rol)6790760(rol) 角色名称
(met)all(met) 全体成员
(met)here(met) 在线成员
(emj)kooknet-logo(emj)[1591057729615250/E8fy7hpRFU1jk1jk] kooknet-logo
