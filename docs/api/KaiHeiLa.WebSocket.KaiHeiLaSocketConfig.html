    public class KaiHeiLaSocketConfig : KaiHeiLaRestConfig
    Remarks

    This configuration, based on KaiHeiLaRestConfig, helps determine several key configurations the socket client depend on. For instance, message cache and connection timeout.

    Examples

    The following config enables the message cache and configures the client to always download user upon guild availability.

    var config = new KaiHeiLaSocketConfig
{
    AlwaysDownloadUsers = true,
    MessageCacheSize = 100
};
var client = new KaiHeiLaSocketClient(config);
