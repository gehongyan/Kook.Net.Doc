ImmutableDictionary<IUser, IReadOnlyCollection<BoostSubscriptionMetadata>>

Remarks

warning

当 AlwaysDownloadBoostSubscriptions 为 true 时，Bot 启动时会自动下载所有服务器的所有助力信息。否则，此属性将为 null。调用 DownloadBoostSubscriptionsAsync(RequestOptions?) 也可以立即下载服务器的所有助力信息，下载完成后，此属性值将被设定。

bool

Remarks

note

此属性仅对基于网关连接的客户端有意义。

此属性为 true 表示，此服务器实体已完整缓存基础数据，并与网关同步。
