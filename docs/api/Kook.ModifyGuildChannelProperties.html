CategoryId

获取或设置要设置到此频道的所属分组频道的 ID。

public ulong? CategoryId { get; set; }

Property Value

ulong?

Remarks

设置此值为某分组频道的 ID 将会使当前频道移动至该分组频道下；设置此值为 0 将会使当前频道脱离其当前所属的分组频道， 位于所有分组频道的上方；如果此值为 null，则当前频道的所属分组频道不会被修改。

Position

获取或设置要设置到此频道的新位置。

public int? Position { get; set; }

Property Value

int?

Remarks

更小的数值表示更靠近列表顶部的位置。设置为与同分组下的其他频道相同的值，将会使当前频道排列于与该频道相邻更靠近列表顶部的位置。 如果此值为 null，则频道的位置不会被修改。
