value string

按钮的值。

click ButtonClickEventType

按钮的点击事件类型。

Remarks

如果 click 设置为 ReturnValue， 则在用户点击按钮时，KOOK 会通过网关下发按钮点击事件，并携带 value 的值。
如果 click 设置为 Link， 则在用户点击按钮时，KOOK 会将用户重定向到 value 指定的 URL。
