    IAsyncEnumerable<IReadOnlyCollection<IUser>>

    Paged collection of users.
    Remarks

    important

    The returned collection is an asynchronous enumerable object; one must call FlattenAsync<T>(IAsyncEnumerable<IEnumerable<T>>) to access the individual messages as a collection.

    This method will attempt to fetch all users that is able to view this channel or is currently in this channel. The library will attempt to split up the requests according to and MaxUsersPerBatch. In other words, if there are 3000 users, and the MaxUsersPerBatch constant is 50, the request will be split into 60 individual requests; thus returning 60 individual asynchronous responses, hence the need of flattening.
